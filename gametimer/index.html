Game Timer

Game timer is a Minecraft mod which allows you to monitor the time you spend in a game session. For myself it is useful as I often simply don't realize that hours are passing by.

Notice

In version 0.3 the save folder is moved.
If you have old data that you wish to keep, simply start the game with the new version, join a world, close the game again, then transfer the content of "<game directory>/saves/GameTimer/" to "<game directory>/gametimer/".

All bug reports go to @Hanse00 on Twitter for now.

Version

The mod is available for 1.6.4 and 1.7.10, it should work with 1.7.2 as well, but that is not supported.

Features

  • Automatic in game notification of how long you've played
  • Configurable notification interval
  • Completely client side - The server doesn't need to even know you have the mod installed
  • Small CO2 footprint - I promise I didn't pollute too much while writing the code

Config

config {
    B:"Debug Mode (Print time to console)"=false
    B:"Display total time"=true
    I:"Notification Interval (In seconds)"=300
    I:"Save interval"=600
}
  • The debug value will just spam your console every second (If you for some reason other than me debugging the code would want that?)
  • Display total time determins if the total time on the server will be displayed together with the session time
  • The notification interval is time time between every in game chat message displaying your game time
  • Save interval is the interval between which the mod saves your total time on the server to disk
